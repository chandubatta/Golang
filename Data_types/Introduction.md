DATA TYPES
* Specific the type of the data that valid GO VARIABLE can hold.
* In the Go language data types are devided into Four types.
    1. Basic type
        ->Numbers
            ^ int8     ^ unt8     ^ float32   ^ complex64   ^ byte  ^ rune ^ uintptr
            ^ int16    ^ uint16   ^ float64   ^ complex128
            ^ int32    ^ uint32
            ^ int64    ^ uint64
        ->Strings
        ->Boolean
    2. Aggregate type
        ->Array
        ->Structure(struct)
    3. Reference type
        ->Pointers
        ->Maps
        ->slices
        ->Functions
        ->Channels
    4. Interface type
