INTRODUCTION
-------------------
* Go version V1.25 (Aug/2025), latest Go V1.26 (Feb,2026).
* For learn Golang Boot.dev's
* The program is saved with .go extension and it is a UTF-8 text file.
* Go language is a STATICALLY typed COMPILER based programming language.
* It is the COMBINATION of C++ and Python. Python is a simple Syntax and C++ is a faster language.
* GO tool can RUN directly, Without VIRTUAL MACHINE.(like JVM)
* Go developed by GOOGLE in 2007 (Robert Giesemer, Rob Pike, Ken thompson)and made OPEANSOURCE with GO V1.0 in 2009.
* Executables are different for OS. (If you developed one application, that application different file for OS to OS). Same application LINUX for different file and WINDOWS for the different file similarly MAC.
* Infra tools like DOCKER, KUBERNETES, DGRAPH and  COCKROACH DB made up of Golang.
* Go supports for CONCURRENCY, which is ability to run multipule tasks simultaneously.
* CONCURRENCY is achieved in Go through the use of GOROUTINES and CHANNELS.
* &nbsp;MISSING  ->"TRY and CATCH" missing we never need in this language.
